# keenetic_tui

Bubble Tea TUI for Keenetic Deck.

## Run

```bash
cd apps/keenetic_tui
go run .
```

## Features

- router add/edit/delete/select
- selected router status and refresh
- client search plus policy/block/Wake-on-LAN actions
- policy-grouped client management
- WireGuard peer inspection
- local-device MAC matching
- auto-refresh toggle and storage diagnostics
